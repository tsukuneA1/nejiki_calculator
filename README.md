ポケモンプラチナ/HGSSで登場する施設バトルファクトリーの通称'金ネジキ'で使えるダメージ計算機です。

## 使い方

# 使用説明書
  ## はじめに
  このツールは、ポケモンのダメージ計算を行うためのものです。ポケモンの技や特性、持ち物などを設定し、ダメージを計算することができます。
  ## 使用方法
  1. ポケモンを選択します。
  2. 技を選択します。
  3. 特性を選択します。
  4. 持ち物を選択します。
  5. レベルを選択します。
  6. ダメージを計算します。
