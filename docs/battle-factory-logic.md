## 相手ポケモンの抽選ロジック

ファクトリーの相手生成はランダム抽選方式。トレーナーIDごとの固定ポケモン対応表は存在しない。
トレーナーID, 難易度が返すのは
1. 抽選プールのID範囲
2. IV値(個体値)
のみ

| トレーナーID | 引くpokemon group | IV(個体値) |
| --- | --- | --- |
| <100 | group1 | 0 |
| <120 | group2 | 4 |
| <140 | group3 | 8 |
| <160 | group4 | 12 |
| <180 | group5 | 16 |
| <200 | group6 | 20 |
| <220 | group7 | 24 |
| <300 | group4~8(ID351-950) | 31 |
| =309 | group4(351-486) | 12 |
| =310 | group4(351~486) | 31 |

各戦のトレーナーIDはどこから来るか
トレーナーIDは難易度Lvを元に抽選。難易度Lv = 連勝数/7
各周回の7戦目だけ一段上の帯となる
| 周回(連勝) | 通常戦(1-6戦目)の相手 | 7戦目(ボス枠)の相手 |
| --- | --- | --- |
| 1周(0-6) | group1 / IV0 | group2 / IV4 |
| 2周(7-13) | group1-2 / IV0-4 | group3 / IV8 |
| 3周(14-20) | group2-3 / IV4-8 | group4 / IV12 |
| 4周(21-27) | group3-4 / IV8-12 | group5 / IV16 |
| 5周(28-34) | group4-5 / IV12-16 | group6 / IV20 |
| 6周(35-41) | group5-6 / IV16-20 | group7 / IV24 |
| 7周(42-48) | group6-7 / IV20-24 | group4-8 / IV31 |
| 8周~(49+) | group7-最終 / IV24-31 | group4-8 / IV31 |

  https://github.com/pret/pokeplatinum/blob/main/src/overlay104/ov104_0223A7F4.c
- 相手ポケモンの生成フロー（`ov104_0223AB0C` / 2戦目以降の `ov104_02234570` → `ov104_0223A918` でランダム抽選）
  https://github.com/pret/pokeplatinum/blob/main/src/overlay104/ov104_022339B4.c
- 抽選結果から実際のポケモン（種族・技・IV・EV・特性）を確定する `ov104_0222DD6C`（IVは `param4` を全ステータス共通でセット、EVは `evFlags` の振り先数で 510 を均等配分、特性は `personality & 1` の2択）
  https://github.com/pret/pokeplatinum/blob/main/src/overlay104/ov104_0222DCE0.c

> 注: IVを `[3,6,9,…]` で返す `ov104_0222E3A8` はバトルタワー/キャッスル系（`ov104_02237DD8.c` / `ov104_022361B4.c`）専用で、**ファクトリーでは使われない**。ファクトリーのIVは上記テーブルの `[0,4,8,12,16,20,24,31]`。

## 自ポケモンの周回初めの抽選ロジック

2つのパラメータによって決定する。パラメータの一つは周回数で、もう一つは仮にparam1とする。
param1は以下の条件で1インクリメントされる
1. 連勝数が+1される
2. 対戦勝利後自ポケモンと相手のポケモンを交換する
敗退すると0に戻る。

param1は閾値を持ち{6,13,20,27,34}でゲームロジックに影響する
具体的に下記のようになる
| param1 | レンタル6匹の中身 |
| --- | --- |
| 0-6 | (基本グループ・個体値) * 6 |
| 7-13 | (基本グループ・個体値) * 5 + (基本グループ・個体値から一段強い) * 1 |
| 14-20 | (基本グループ・個体値) * 4 + (基本グループ・個体値から一段強い) * 2 |
| 21-27 | (基本グループ・個体値) * 3 + (基本グループ・個体値から一段強い) * 3 |
| 28-34 | (基本グループ・個体値) * 2 + (基本グループ・個体値から一段強い) * 4 |
| 35~ | (基本グループ・個体値) * 1 + (基本グループ・個体値から一段強い) * 5 |

基本グループ・個体値は周回数で決定する。具体的に下記のようになる
| 周回(連勝) | レンタル基本(6匹の土台) | 末尾 v6匹の強化先 |
| --- | --- | --- |
| 1周(0-6) | group1 / IV0 | 強化なし(param1が0) |
| 2周(7-13) | group2 / IV4 | group3 / IV8 |
| 3周(14-20) | group3 / IV8 | group4 / IV12 |
| 4周(21-27) | group4 / IV12 | group5 / IV16 |
| 5周(28-34) | group5 / IV16 | group6 / IV20 |
| 6周(35-41) | group6 / IV20 | group7 / IV24 |
| 7周(42-48) | group7 / IV24 | 最終(351~950) / IV31 |
| 8周~(49+) | 最終(351~950) / IV31 | 強化なし(既に最強) |
