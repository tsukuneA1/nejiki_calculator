# 金ネジキとは？ バトルファクトリー49連勝の基本ルール

「金ネジキ」とは、『ポケットモンスター プラチナ』と『ポケットモンスター ハートゴールド・ソウルシルバー』（以下、HGSS）のバトルファクトリーで、49戦目に登場するファクトリーヘッド「ネジキ」の通称です。

また、プレイヤーの間では、金ネジキを倒すまでの**49連勝に挑む遊びそのもの**も「金ネジキ」と呼ばれます。自分で育てたポケモンは使わず、施設から借りたポケモンだけで7周を勝ち抜くのが、この挑戦の特徴です。

この記事では、これから金ネジキに挑戦する人に向けて、次の内容を順番に説明します。

- 金ネジキとバトルファクトリーの関係
- 挑戦開始から49戦目までの流れ
- レンタル、対戦、交換のルール
- 周回によるポケモンの変化
- 『プラチナ』とHGSSの違い

## 金ネジキとバトルファクトリー

### 金ネジキは49戦目のネジキ

ネジキは、バトルファクトリーを担当するフロンティアブレーンです。シングルバトルを連勝すると、次の2回に対戦相手として登場します。

| 対戦 | 登場するタイミング | 勝利の証 |
| --- | ---: | --- |
| 銀ネジキ | 21戦目（3周目の7戦目） | シルバープリント |
| 金ネジキ | 49戦目（7周目の7戦目） | ゴールドプリント |

ネジキの手持ちもレンタルポケモンから選ばれるため、毎回同じ3匹が登場するわけではありません。決まった手持ちへの対策ではなく、その場で判明した相手に合わせて行動する必要があります。

### バトルファクトリーはレンタルポケモンで戦う施設

バトルファクトリーは、さまざまな対戦施設が集まる「バトルフロンティア」の一つです。通常の冒険とは異なり、プレイヤーが捕まえたり育てたりしたポケモンは使用しません。

挑戦中に使うのは、種類、能力、性格、技、特性、持ち物があらかじめ決められたレンタルポケモンです。手持ちを育成する準備がいらない一方で、提示された候補から戦える3匹を選ぶ判断力が求められます。

## 49戦目までの流れ

金ネジキへの挑戦は、7戦を1周として進みます。7周目の最後に金ネジキを倒すと49連勝です。

| 周回 | 通算の対戦数 | その周の7戦目 |
| ---: | ---: | --- |
| 1周目 | 1〜7戦目 | 通常の対戦相手 |
| 2周目 | 8〜14戦目 | 通常の対戦相手 |
| 3周目 | 15〜21戦目 | 21戦目に銀ネジキ |
| 4周目 | 22〜28戦目 | 通常の対戦相手 |
| 5周目 | 29〜35戦目 | 通常の対戦相手 |
| 6周目 | 36〜42戦目 | 通常の対戦相手 |
| 7周目 | 43〜49戦目 | 49戦目に金ネジキ |

1周の進行は、次の4段階に分かれます。

1. 施設から提示された6匹を確認し、最初の3匹を選ぶ
2. スタッフから次の相手に関する情報を聞く
3. 3対3のバトルを行う
4. 勝利後、必要なら相手のポケモン1匹と自分のポケモン1匹を交換する

2〜4を繰り返して7勝すると、その周は終了です。次の周では、それまで使っていた3匹を返却し、新しく提示される6匹から3匹を選び直します。途中で負けた場合は連勝が止まり、次の挑戦は1戦目からやり直しです。

## 挑戦中のルール

### 対戦形式

バトルファクトリーにはシングル、ダブル、マルチの各形式があります。このうち、21戦目と49戦目にネジキが登場するのはシングルバトルです。

レベルは「レベル50」と「オープンレベル」から選びます。第四世代のバトルファクトリーでは、オープンレベルのレンタルポケモンはレベル100になります。どちらも金ネジキまでの目標は49連勝ですが、登場するポケモンの範囲や型が周回ごとに異なります。

### 最初のレンタル

各周の開始時に6匹の候補が提示され、その中から3匹を選びます。同じ種類のポケモンや同じ持ち物が候補内で重複しないように選ばれます。

### 対戦前に得られる情報

各対戦の前には、スタッフが次の相手についてヒントを教えてくれます。序盤はポケモン名まで分かりますが、連勝するほど情報は少なくなります。

| 周回 | シングルバトルで得られる情報 |
| ---: | --- |
| 1周目 | 相手の3匹すべて |
| 2周目 | 相手の先頭から2匹 |
| 3周目 | 相手の先頭1匹と、そのポケモンの一つ目の技 |
| 4周目 | 相手の先頭ポケモンの一つ目の技 |
| 5周目以降 | 相手3匹に最も多いタイプ |

後半の周回ほど、対戦前に分かる相手の情報は少なくなります。

### 勝利後の交換

勝利後は、相手が使っていた3匹のうち1匹を、自分の3匹のうち1匹と交換できます。交換しないことも可能です。

交換には二つの目的があります。

1. その周を勝ち抜きやすい手持ちに改善する
2. 交換回数を増やし、次の周の最初に混ざる強い候補の数を増やす

交換回数には、各挑戦の最初に行うレンタルも1回として含まれます。累計回数が一定の値に達すると、次の周の6匹に、一つ先の周相当のポケモンが混ざります。

| 累計レンタル・交換回数 | 次の周の6匹に混ざる強い候補 |
| ---: | ---: |
| 0〜6回 | 0匹 |
| 7〜13回 | 1匹 |
| 14〜20回 | 2匹 |
| 21〜27回 | 3匹 |
| 28〜34回 | 4匹 |
| 35回以上 | 5匹 |

## 周回で変わるレンタルポケモン

### 型はあらかじめ決まっている

バトルファクトリーのポケモンは完全なランダム生成ではありません。同じ種類でも、性格、努力値、技、持ち物が異なる複数の型が用意されており、周回に対応した候補から抽選されます。

相手のポケモン名だけで型を一つに断定できない場合があります。周回、見えた技、持ち物の発動などを組み合わせると、候補を絞り込めます。

### 個体値は周回ごとに上がる

レンタルポケモンと通常の対戦相手は、すべての能力に同じ個体値が設定されます。基本となる個体値は周回が進むほど高くなります。

| 周回 | 全能力共通の個体値 |
| ---: | ---: |
| 1周目 | 0 |
| 2周目 | 4 |
| 3周目 | 8 |
| 4周目 | 12 |
| 5周目 | 16 |
| 6周目 | 20 |
| 7周目 | 24 |
| 8周目以降 | 31 |

各周の7戦目には、原則として次の周に相当する強さの相手が出ます。金ネジキのポケモンは個体値31です。そのため、7周目の自分の初期レンタルより金ネジキの方が、同じ種類・同じ型でも能力で有利になる場合があります。

出現する型、能力値、技、特性、持ち物は[金ネジキのポケモン一覧](/poke-search)で検索できます。

## 『プラチナ』とHGSSの違い

49連勝、レンタル、勝利後の交換といった基本ルールは共通です。一方、ネジキが抽選するポケモンの範囲には作品とレベル設定による違いがあります。

特に日本版『プラチナ』では抽選処理の都合により、レベル50の金ネジキはHGSSより早い周回相当の型を使います。反対に、オープンレベルの銀ネジキはHGSSより遅い周回相当の型を使います。

| 対戦 | レベル | 『プラチナ』 | HGSS |
| --- | --- | --- | --- |
| 銀ネジキ | レベル50 | 早い周回相当の型 | 早い周回相当の型 |
| 銀ネジキ | オープンレベル | 遅い周回相当の型 | 早い周回相当の型 |
| 金ネジキ | レベル50 | 早い周回相当の型 | 遅い周回相当の型 |
| 金ネジキ | オープンレベル | 遅い周回相当の型 | 遅い周回相当の型 |

したがって、同じ「金ネジキ」でも、作品とレベル設定が違えば登場する可能性のある型の範囲も変わります。
