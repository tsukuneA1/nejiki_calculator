# 金ネジキとは

<img width="174" height="289" alt="Image" src="https://github.com/user-attachments/assets/e95739f3-620e-494f-a89e-9d5b00ff4bde" />

金ネジキとは「ポケットモンスター プラチナ」および「ポケットモンスター ハートゴールド・ソウルシルバー」の対戦施設「バトルファクトリー」における対戦施設で49人目に登場するファクトリーヘッド「ネジキ」のことです。「金ネジキ」は俗称としてバトルファクトリー対戦施設を指すことが多いです。

レンタルポケモンのみを使い、7連勝 x 7周を達成することを目指します。

ゲーム自体は「プラチナ」の発売が2008年ですから、20年弱ほど前のレトロゲームと呼べるのですが、「加藤純一」(*1)さんなどをはじめ配信者にプレイされることが多く、その難易度も相まって未だにプレイヤーが存在するコンテンツとなっています。
*1 https://www.youtube.com/watch?v=uO7YeZEA0XM
